# Using AI

## Project Purpose

Using AI is Luke and Ashlar's practical programme for learning how to work with ChatGPT at the highest useful standard.

The project is not limited to writing clever prompts. Its purpose is to develop the knowledge, judgement, habits, prompts, evaluation methods, and repeatable workflows needed to think and work better with AI.

## Mission

To develop the knowledge, habits, and workflows that enable a human and AI to consistently produce work that is clearer, more accurate, more creative, and more valuable than either could produce alone.

## Intended Outcomes

The project should help Luke:

- produce better work;
- save hours every week;
- become a better thinker and decision-maker;
- use AI as a deep tutor, accountable coach, researcher, strategist, and collaborative thought partner;
- continue developing as an all-round more capable person.

## Working Philosophy

1. Pursue understanding, not merely answers.
2. Prefer truth over confident-sounding speculation.
3. Separate established facts, reasonable inferences, educated estimates, and unknowns.
4. Refine important ideas through probing questions and constructive alternatives.
5. Challenge assumptions when they become material to the work, without disrupting open brainstorming unnecessarily.
6. Treat prompting as an iterative collaboration rather than a search for one magical instruction.
7. Use AI to enhance human judgement, not replace it.
8. Preserve important knowledge in the repository so progress accumulates across conversations.

## Core Workflow

**Prompt → Response → Evaluation → Refined Prompt → Improved Response → Reusable Principle**

Every useful discovery should be tested for general lessons that can be added to the handbook or prompt library.
