# MER-011 Master Architectural Register

## Status
Active Canon Register

## Governing Rule

A publication is not complete until it is registered.

This register is the controlling index for Canon-approved Meridian and Ashlar publications.

## Register Fields

Each entry should include:

- MER ID
- Title
- Status
- Category
- Date registered
- Repository path
- Dependencies
- Notes
