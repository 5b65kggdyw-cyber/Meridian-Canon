# Meridian Canon

The Meridian Canon is the central repository for the architectural, editorial, constitutional, and implementation work of the Meridian and Ashlar project.

This repository exists to preserve continuity, prevent loss of important work, and create a stable foundation for all future publications, decisions, systems, and build packs.

## Core Rule

A publication is not complete until it is registered.

Every Canon-approved publication must be entered into `MER-011 Master Architectural Register` before it is considered complete.

## Current Foundation

- Repository Constitution
- Master Architectural Register
- Editorial Process
- Canon Publications directory
- Legacy Recovery directory
- Implementation Packs directory

## Stewardship

Ashlar is treated as the repository steward: responsible for consistency, recovery, refinement, continuity, and realistic next-step guidance.

The project begins with the user and Ashlar as the working pair. Scalability may be considered later, but the present architecture should serve this immediate partnership first.
