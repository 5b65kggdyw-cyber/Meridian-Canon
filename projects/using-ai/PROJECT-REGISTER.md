# Using AI Project Register

This register records the formal publications and major revisions of the Using AI project.

## Completion Rule

A major Using AI publication is not complete until it is entered into this register.

Publications that establish or alter wider Meridian Canon architecture should additionally be recorded in the Canon's master architectural register.

## UAI-001 — Using AI Project Foundation

- **Status:** Published
- **Publication date:** 16 July 2026
- **Repository path:** `projects/using-ai/README.md`
- **Purpose:** Establish the project's mission, intended outcomes, working philosophy, core workflow, structure, and stewardship.
- **Key principles introduced:**
  - Understanding rather than answers alone.
  - Truth over confident speculation.
  - Explicit distinction between facts, inferences, estimates, and unknowns.
  - Constructive refinement through probing questions.
  - AI as an enhancer of human judgement.
  - Repository preservation for continuity.
- **Dependencies:** Meridian Canon repository constitution and editorial methodology.
- **Next review trigger:** A material change to project scope, governance, or intended outcomes.
