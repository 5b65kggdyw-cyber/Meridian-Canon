# Founder's Architectural Register Migration Map

## Status
Active Migration Specification

## Source
Image supplied by the founder showing the Ashlar Meridian Master Architectural Register - Founder's Edition.

## Purpose

This document converts the visual Founder's Register into a repository migration map.

The register is to be treated as an authoritative migration specification for all current Canon, legacy/pre-Canon, and planned future Canon publications shown in the image.

## Governing Rule

Every document named in the Founder's Register must have one of the following repository outcomes:

1. Migrated as a Canon publication
2. Archived as a Legacy publication
3. Republished into a new Canon publication
4. Tracked as a planned roadmap item
5. Retired with explanation

No listed document should remain unaccounted for.
